# 🌌 Desenrola AI

A **DesenrolaAI** nasceu para traduzir código em resultados reais. Este projeto é a plataforma oficial da DesenrolaAI, construída com uma estética de alta performance, técnica e sofisticada, utilizando o sistema de design **Obsidian Aura**.

---

## ✨ Essência do Design: Obsidian Aura

O **Obsidian Aura** é uma linguagem de design de alta performance que evoca precisão, inteligência e modernidade. 

- 🌑 **Profundidade**: Um canvas azul-escuro profundo (`Obsidian`) serve como base.
- 💡 **Bioluminescência**: Acentos em azul e lima criam um ambiente premium e tecnológico.
- 📐 **Geometria Suave**: O uso de bordas arredondadas e glassmorphism traz uma abordagem amigável e sofisticada.

---

## 🚀 Funcionalidades Principais

- 💎 **Interface Glassmorphism**: Camadas elevadas com transparência e desfoque de fundo.
- ⚡ **Performance Blue**: Call-to-actions (CTAs) otimizados com gradientes vibrantes para máxima conversão.
- 📱 **Responsividade Total**: Design fluido que se adapta de smartphones a monitores ultrawide.
- 🎭 **Animações Atmosféricas**: Transições Snappy e suaves que guiam o olhar do usuário.

---

## 🛠️ Stack Tecnológica

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Biblioteca Base**: React 19
- **Estilização**: Modern Vanilla CSS (utilizando Custom Properties do Design System)
- **Tipografia**: 
  - `IBM Plex Serif` (Headings Display)
  - `League Spartan` (Headings Functional)
  - `Archivo` (Body & UI)
- **Linguagem**: TypeScript

## 💻 Como Iniciar o Desenvolvimento

### Pré-requisitos
- Node.js (versão recomendada: >= 20.x)
- Gerenciador de pacotes (npm, yarn ou pnpm)

### Instalação

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/seu-usuario/site-desenrolaai-next.git
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Inicie o servidor local:**
   ```bash
   npm run dev
   ```

4. **Explore:**
   Acesse [http://localhost:3000](http://localhost:3000) e veja a magia acontecer.
